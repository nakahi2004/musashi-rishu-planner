# 武蔵履修プランナー

武蔵大学生向けの履修登録サポートツールです。
科目一覧の閲覧・優先度管理・カレンダー確認・単位状況の把握を、ブラウザ上でオフラインで行えます。

---

## デモ

まずは試してみたい方へ：**`rishu_planner_demo.html`** をダウンロードしてブラウザで開くだけで動作します（ファイルのアップロード不要）。

---

## 機能

- **リスト表示** — 大分類・中分類・小分類で科目をツリー表示。科目名・ナンバリングで検索可能
- **優先度設定** — 科目ごとにA〜Dの優先度とメモを記録（ブラウザに自動保存）
  - `A` 絶対受けたい　`B` 受けたい　`C` 選択肢には入れとく　`D` 絶対受けたくない
- **カレンダー表示** — 曜日・時限を設定した科目をカレンダーで確認。優先度ごとのフィルタリング対応
- **単位状況タブ** — 取得済み・要件単位を分類ごとに一覧表示（成績PDFを使用した場合）
- **シラバスリンク** — タイルから3Sのシラバス検索ページに直接アクセス（科目名を自動コピー）
- **オフライン動作** — 一度開けばネット接続不要。データはすべてブラウザのlocalStorageに保存

## 使い方

### ① CSVを生成する（Google Colab）

以下の2つのPDFを用意してください。

| ファイル | 入手先 |
|---|---|
| 履修要項PDF（学科名が入ったもの） | 大学からダウンロードされたフォルダ内 |
| 成績PDF | [3S](https://3s.musashi.ac.jp/) → 成績タブ → 成績照会 → 右上「PDF」 |

1. [generate_csv.ipynb](./generate_csv.ipynb) を Google Colab で開く
2. **ステップ1〜4** を順番に実行
3. `merged_courses.csv` がダウンロードされる

> **成績PDFをアップロードしたくない場合（新1年生など）**
> ステップ3をスキップしてステップ4を実行すると、履修済みマーク・単位状況なしの
> `merged_courses_no-personal-data.csv` が生成されます。

### ② Webアプリで開く

1. `rishu_planner.html` をブラウザで開く
2. 生成した CSV をドラッグ＆ドロップ、または貼り付けで読み込む
3. 履修計画を立てる
